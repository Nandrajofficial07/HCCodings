LIST   (li)

Definitions

  • Use of list is in HTML to sequence items.

  • list is written term of "< li >(List item)"in HTML.

  • Ther are THREE types of List.

  • Ordered List
  • Unordered List
  • Definitions Or Discription List

    • Note:- Ordered and Unordered list has a child tag "Li"(List Item) and
    Definition list has a child tag "Dt"(Description or detail Term) and
    it has also a child tag "Dd"(Describe the term in a Description list).
    Always remmeber that without child tags OL,UL and DL don't work.

  • Li= item list
  • Ol=ordered list
  • Ul=unordered list
  • Dl=Definition or Description list
  • Dt=detail or description term
  • Dd=Describe the term in a Description list

    • Ordered List   (Ol)

  • By default ordered list print numerical value ordered vise at the starting of each item in the list.

  • Ordered list is written term of "< ol >"in HTML..

  • With ordered list we are able to change numerical value into alphabates of any language through the property of css.
